# WanAndroid客户端

## 简介

这是一款分享优质技术文章的APP，此项目使用[WanAndroid](https://www.wanandroid.com)网站提供的API，基于`MVP` + `Butterknife` + `Rxjava` + `Retrofit` + `Dagger2` + `GreenDao` + `Glide` + `Material Design`等架构实现。
